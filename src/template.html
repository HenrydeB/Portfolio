ShotMark

E-Commerce page dedicated to the resale and trading of film, photo, and sound equipment. Think PoshMark, but for film equipment. Using the MERN stack to store product data in MongoDB, connecting the information to the server and using express to communicate the information to the front end. React and Redux are used to design the user interface, and provide functionality such as adding product to cart, loading items to product page, and changing/resetting the number of cart items.

Smart-Brain

This is a web application that uses an external face identification API! Users register, their information is logged in a PostgreSQL database, and the user is then able to submit image URLs from the internet. The application then uses the Clarifai facial recogniton API to identify any faces in the image while keeping track of the number of submissions. Give it a try! Smoke Signals: Wildfire Tracker

Application that uses Google Maps API and the Nasa Earth Observatory Natural Event Tracker API data to actively plot and track wildfires around the world. Application tracks the number of wildfires around the world using live coordinates provided by NASA satellites. This can be expanded to include other natural events such as storms and glacial movement.
